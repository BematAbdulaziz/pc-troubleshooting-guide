# 💻 PC Troubleshooting Guide

## 🔍 Overview

This repository contains common computer issues and their step-by-step solutions based on real-world IT support experience.

---

## ⚠️ Issue 1: Computer Not Turning On

### 🔧 Steps:

1. Check power cable connection
2. Try different power socket
3. Inspect SMPS fan (is it spinning?)
4. Remove RAM → clean → reinsert
5. Disconnect all peripherals and try again

👉 If still not working → possible SMPS or motherboard issue

---

## 🐢 Issue 2: Slow Performance

### 🔧 Steps:

1. Press `Ctrl + Shift + Esc` → Open Task Manager
2. Check CPU / RAM usage
3. Disable startup apps
4. Run Disk Cleanup
5. Upgrade RAM (if below 8GB recommended)

---

## 🖥️ Issue 3: No Display

### 🔧 Steps:

1. Check monitor power
2. Reconnect HDMI/VGA cable
3. Try another monitor
4. Remove and reinsert RAM
5. Check GPU connection

---

## 🌐 Issue 4: No Internet

### 🔧 Steps:

1. Restart router
2. Run `ipconfig /flushdns` in CMD
3. Disable/Enable network adapter
4. Update network drivers
5. Try LAN cable

---

## 🧠 Tip

Always isolate the problem step-by-step before replacing hardware.
